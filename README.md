# Salary Calculator App

A Flutter-based mobile application for calculating and managing salary computations with encryption support for secure data handling.

## Features

- 💰 **Salary Calculation**: Calculate salaries with various parameters
- 🔒 **Data Encryption**: Secure sensitive salary information with built-in encryption
- 📱 **Cross-Platform**: Runs on Android, iOS, macOS, Windows, Linux, and Web
- 🎨 **User-Friendly Interface**: Clean and intuitive UI for easy navigation
- 📊 **Fast Computation**: Quick and efficient salary calculations

## Prerequisites

- [Flutter](https://docs.flutter.dev/get-started/install) (latest stable version)
- [Dart SDK](https://dart.dev/get-dart)
- Android Studio / Xcode (for mobile development)
- A code editor (VS Code, Android Studio, or IntelliJ)

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/salary_calc_app.git
   cd salary_calc_app
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```
